Política de Privacidade

Esta Política de Privacidade descreve como as informações são coletadas, utilizadas e protegidas ao utilizar este site.

1. Dados Coletados

Podemos coletar informações fornecidas voluntariamente pelo usuário, como nome no jogo, identificadores de conta, informações de contato e dados relacionados a pagamentos. Além disso, dados técnicos básicos podem ser coletados automaticamente, incluindo endereço IP, data, horário, navegador, sistema operacional e origem da navegação.

2. Uso das Informações

As informações coletadas são utilizadas exclusivamente para processamento de pedidos, verificação de pagamentos, prevenção de fraudes, comunicação com o cliente e melhoria da segurança do serviço.

3. Compartilhamento de Dados

Este site não vende, aluga ou compartilha dados pessoais com terceiros para fins comerciais. Alguns dados podem ser processados por serviços externos estritamente necessários ao funcionamento do site, como provedores de e-mail e plataformas de comunicação.

4. Segurança

Medidas razoáveis são adotadas para proteger as informações contra acesso não autorizado, uso indevido ou divulgação indevida. No entanto, nenhum sistema é completamente seguro.

5. Responsabilidade

O usuário reconhece que o uso deste site é realizado por sua própria conta e risco. Este site não possui vínculo oficial com a desenvolvedora ou publicadora do jogo Where Winds Meet.

6. Alterações

Esta Política de Privacidade pode ser atualizada a qualquer momento, sem aviso prévio, sendo responsabilidade do usuário consultá-la periodicamente.
